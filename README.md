
# Análise de Vendas com Pandas

Projeto desenvolvido com foco em aprendizado prático da biblioteca Pandas,
aplicando conceitos de análise e transformação de dados em um cenário de vendas.

## Objetivo

Demonstrar o uso do Pandas para:

- Limpeza e tratamento de dados
- Padronização de informações
- Criação de métricas de negócio
- Análises exploratórias e agregações
- Merge entre diferentes bases
- Análise temporal de faturamento

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Estrutura dos dados

- vendas_tech.csv: base principal de vendas
- gerentes_lojas.xlsx: base com informações de gerentes e metas

## Principais análises realizadas

- Faturamento por loja
- Ranking de produtos no canal online
- Produtos mais vendidos por loja
- Verificação de atingimento de meta por gerente
- Evolução mensal do faturamento

## Como executar

1. Clone o repositório
2. Instale as dependências:
   pip install pandas numpy
3. Execute o notebook no Jupyter
